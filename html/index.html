Deskripsi Mata Kuliah

Pelajaran Cloud Computing memberikan pemahaman mendalam tentang konsep dan praktik penting dalam komputasi awan. Fokus utama adalah pemanfaatan layanan awan, pengelolaan sumber daya, dan implementasi aplikasi efisien menggunakan GitHub dan Docker.

GitHub digunakan sebagai platform kolaboratif untuk pengembangan perangkat lunak, memungkinkan kerja tim secara efisien dan menyediakan kontrol versi yang kuat. Siswa akan mempelajari cara mengelola kode sumber, berkolaborasi dengan sesama pengembang, dan menggunakan alat GitHub untuk pengecekan kode (code review) dan manajemen proyek.

Docker, sebagai alat kontainerisasi, menjadi fokus utama dalam mengemas, mendistribusikan, dan menjalankan aplikasi secara konsisten di lingkungan awan. Siswa memahami konsep dasar kontainer, membuat dan mengelola Docker images, serta mengintegrasikan kontainer ke dalam alur kerja pengembangan dan penyebaran aplikasi.

Melalui kombinasi GitHub dan Docker, pelajaran ini membekali siswa dengan keterampilan praktis dalam mengembangkan, menguji, dan menyebarkan aplikasi secara efektif di lingkungan komputasi awan. Siswa mampu membuat proses pengembangan lebih kolaboratif dan pengelolaan aplikasi lebih efisien, menciptakan landasan yang kuat untuk bekerja dalam industri teknologi yang terus berkembang.
